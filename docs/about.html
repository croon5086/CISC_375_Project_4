Description of the tools (frameworks, APIs, etc.) used to create the application

Leaflet:
The Leaflet API allows us to render the interactive map of St. Paul. Also it allowed us to overlay day on top of the map. The map was able to be modified by giving it coordinates and panning, and zooming.
Nominatim:
The Nominatim API allowed us to search for locations by name, and also retrieve a location's name based on lat/long coordinates.
